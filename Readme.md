# CTF Writeups and Documentation

Repository ini berisi arsip writeup, solver, catatan, dan file pendukung dari berbagai kompetisi Capture The Flag (CTF) yang pernah dikerjakan.

Dokumentasi di dalam repository ini mencakup berbagai challenge dari beberapa event, mulai dari writeup final, script eksploitasi atau solver, file challenge, hasil ekstraksi, sampai catatan singkat selama proses penyelesaian.

## Kategori

Beberapa kategori yang ada di repository ini:

- Cryptography
- Pwn
- Reverse Engineering
- Web Exploitation
- Miscellaneous
- Forensics
- Steganography
- OSINT atau challenge berbasis analisis file

## Isi Umum

Jenis file yang umum ditemukan di repository ini:

- `solver.py`, `solve.py`, atau script serupa untuk otomatisasi penyelesaian challenge
- File challenge asli seperti binary, archive, gambar, audio, atau packet capture
- Screenshot dan catatan analisis
- Writeup PDF untuk dokumentasi final
- Output hasil ekstraksi atau hasil decoding

## Catatan

Repository ini dibuat sebagai dokumentasi pembelajaran dan arsip pribadi/team dalam menyelesaikan challenge CTF. Beberapa solusi mungkin masih bersifat eksploratif dan mengikuti kondisi challenge saat kompetisi berlangsung.
